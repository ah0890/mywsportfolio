# 📚 Books Scraper (Web Scraping Project)

A simple Python-based web scraping script that extracts book data (name, price, and link) from a demo website and saves it into a CSV file.

---

## 🔗 Repository Link

👉 https://github.com/ahaulakh0890/myfwspro

---

## 🚀 Features

 📖 Scrape book titles from website\
 💲 Extract book prices\
 🔗 Collect product links\
 📁 Save data into CSV file\
 ⚡ Fast scraping using requests (no browser needed)

---

## 🛠️ Tech Stack

* Python
* Requests
* BeautifulSoup
* CSV
