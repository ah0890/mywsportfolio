# 🏬 Store Inventory Management System

A simple Python-based inventory management system that allows you to organize and track items by category, product, and quantity.

## 🚀 Features

 📦 Add items with category, product, and quantity\
 🗂️ Organized nested inventory structure\

 📊 Display inventory in a structured format\
 🧠 Simple and beginner-friendly logic

## ⚙️ How It Works

The system organizes inventory in the following hierarchy:

```
Category → Product → Item → Quantity
```

Example:

```
Furniture → Amjad → Chairs → 5
Electronics → Umar → PC → 2
```
